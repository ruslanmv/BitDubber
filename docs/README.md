## Important Notes

### For Parents and Educators

BitDubber uses AI and voice recognition to help children interact with computers. Please note:

- **Supervision Required**: Children should always use BitDubber under adult supervision
- **Internet Safety**: Combine BitDubber with parental controls and content filtering
- **Privacy**: Never allow children to speak personal information into the system
- **Setup**: Adults must configure API credentials before children can use the system
